# Bill Splitter

A simple and mobile-friendly web app to split a bill equitably among a group of people

## Features

- Split any bill among multiple people
- Optional tip calculator
- Transparent receipt-style breakdown
- Input validation with error messages
- Fully responsive, designed for mobile

## Technologies

- HTML5
- CSS3
- JavaScript (Vanilla)

## How to use

1. Enter the total bill amount
2. Enter the number of people
3. Enter a tip percentage (optional)
4. Press **Calculate**
5. The receipt will show the breakdown and the amount each person owes
